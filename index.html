À propos de moi

____________________________

Bonjour, je m'appelle Elmaouaj Achraf, je suis un élève ingénieur marocain actuellement dans la 3éme année cycle d'ingénieur de l'Ecole des Sciences de l'information ( ESI ).

Pendant mon parcours scolaire, j'ai participé et réalisé plusieurs projets dans les domaines suivants: développement web, Data visualization, Data mining et Data warehouse. Ainsi que j'ai effectué deux stages dans des grandes organisations ( Nokia / Poste Maroc), dans lesquelles j'ai eu l'opportunité de découvrir le monde professionnel.

Machine Learning

Il s’agit d’une science moderne permettant de découvrir des patterns et d’effectuer des prédictions à partir de données en se basant sur des statistiques, sur du forage de données, sur la reconnaissances de patterns et sur les analyses prédictives

Data Mining

L’analyse de données depuis différentes perspectives et le fait de transformer ces données en informations utiles, en établissant des relations entre les données ou en repérant des patterns.

75

Data Visualisation

La Data Visualisation est une technique d'exploration et d'analyse des données numériques à l'aide de graphiques.

55

Data Warehouse

Un entrepôt de données est une base de données regroupant une partie ou l'ensemble des données fonctionnelles d'une entreprise. Il entre dans le cadre de l'informatique décisionnelle ; son but est de fournir un ensemble de données servant de référence unique, utilisée pour la prise de décisions dans l'entreprise par le biais de statistiques et de rapports réalisés via des outils de reporting.

50

Développement Web

Le développement Web désigne de manière générale les tâches associées au développement de sites Web destinés à être hébergés via un intranet ou Internet. Le processus de développement web comprend, entre autres, la conception de sites web, le développement de contenu web, l’élaboration de scripts côté client ou côté serveur et la configuration de la sécurité du réseau.
